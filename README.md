# Team Task Manager

A modern team collaboration application built with React, TypeScript, and Firebase that helps teams organize projects, manage tasks, assign work, and collaborate in real time.

## Overview

Team Task Manager is a full-stack frontend application designed to simplify project organization for individuals and teams. Users can create teams, organize projects, assign tasks, invite members, and track work through a clean, responsive interface powered by Firebase's real-time database.

This project was built to strengthen my skills in modern frontend development while focusing on scalable architecture, reusable components, and real-time data synchronization.

## Features

* User authentication with Firebase Authentication
* Team creation and management
* Team invitation system
* Real-time collaboration with Firestore
* Project organization
* Task creation, editing, and deletion
* Task status management
* Task assignment
* Due dates and overdue tracking
* Team member management
* Responsive design for desktop and mobile
* Modern UI with reusable React components
